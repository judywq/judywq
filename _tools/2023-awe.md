## What is {{ page.title }}

A Python program for providing summative assessment of TOEFL independent writing essays. The program is built on a finetuned GPT-3.5 model with official dataset from ETS.
The program currently achieves a QWK of 0.78 and an RMSE of 0.57 against ground truth scores from ETS, rivaling and even surpassing ETSâ€™s e-rater engine.
